# Ray tracer

## Description

MiniRT project for Codam (42 Network).

## Team

Rik and Kwok-po

## usage

git clone (git url)

git submodule update --init

make re (optional)RAYS=n (optional)BOUNCE=n

./minirt path/to/maps/map.rt (optional)width (optional)height
